#### 指令教學
- 使用 `!question <問題>` 進行塔羅占卜
- 使用 `!chat <對話>` 聊天
- 使用 `!complain <抱怨>` 抱怨，AI可能會兇你
- 使用 `!why <問題>` 進行問答
